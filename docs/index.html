灵动刘海交互

启动后自动收起为屏幕顶部中央的小药丸，悬浮或点击即刻展开编辑区域，收回时带有弹性弹簧动画和高斯模糊过渡。

富文本编辑器

基于 TipTap 构建，支持加粗、斜体、删除线、行内代码、引用块、有序/无序列表、任务列表、链接插入、截图粘贴、分割线等。

多页面管理

最多支持 10 个页面，顶部胶囊式导航栏切换，支持拖拽排序，动画平滑无抖动。长按弹出置顶菜单。

个性化设置

深色/浅色/跟随系统主题，多种强调色可选，默认字号调节，唤醒方式切换，开机自启，数据导入导出。

全局快捷键

Ctrl+Alt+Z 直接呼出窗口并展开编辑区域，随时随地快速记录灵感，不打断工作流。

数据持久化

使用 SQLite (sql.js) 本地存储，数据保存在 userData/memo.db。关闭窗口后驻留系统托盘，双击托盘图标即可恢复。
